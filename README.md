#### Aplicacion server

1. En la carpeta principal del proyecto hacer

```sh
> npm install
```

2. Correr la aplicacion servidor

```sh
> npm run start
```

#### Aplicacion cliente

1. En la carpeta app_client hacer

```sh
> npm install
```

2. Correr la aplicacion servidor

```sh
> npm run start
```
